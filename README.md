# AWS Technical Essentials 5.8.2 (Español)

Traducción técnica al español del material **AWS Technical Essentials 5.8.2**, estructurada en formato Markdown y pensada para su consulta desde GitHub y Obsidian.

## Objetivos

- Traducir fielmente el contenido de la guía original.
- Mantener la estructura del curso.
- Representar los diagramas mediante diagramas ASCII.
- Utilizar la terminología oficial de AWS.
- Mantener el contenido en Markdown estándar.

## Estructura

```text
00 - Course Overview/
01 - Introduction to AWS/
02 - AWS Compute/
03 - AWS Networking/
04 - AWS Storage/
05 - Databases/
06 - Monitoring, Load Balancing and Scaling/
Anexos/
```

## Convenciones

- La información adicional no forma parte de esta traducción y se documentará en la carpeta **Anexos**.
- Los diagramas se representan mediante diagramas ASCII.
- Los nombres oficiales de los servicios de AWS no se traducen.
- La estructura sigue el mismo orden que la guía oficial.

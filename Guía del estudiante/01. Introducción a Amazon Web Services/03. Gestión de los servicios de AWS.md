# Gestión de los servicios de AWS

AWS proporciona diferentes herramientas para administrar y utilizar sus servicios. Cada una está orientada a un perfil de usuario y a una forma distinta de interactuar con la plataforma.

---

## AWS Management Console

La **AWS Management Console** es una interfaz gráfica basada en web que permite crear, configurar y administrar los recursos de AWS desde un navegador.

Está diseñada para facilitar la administración de los servicios sin necesidad de utilizar herramientas de línea de comandos.

---

## AWS Command Line Interface (AWS CLI)

La **AWS Command Line Interface (AWS CLI)** permite administrar los servicios de AWS mediante comandos desde un terminal.

Es una herramienta especialmente útil para automatizar tareas y gestionar recursos de forma eficiente.

---

## AWS Software Development Kits (AWS SDKs)

Los **AWS SDKs** permiten integrar los servicios de AWS directamente en aplicaciones desarrolladas en distintos lenguajes de programación.

Cada SDK proporciona bibliotecas y herramientas que simplifican el uso de las API de AWS.

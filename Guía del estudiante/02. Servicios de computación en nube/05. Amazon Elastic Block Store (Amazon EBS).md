# Amazon Elastic Block Store (Amazon EBS)

**Amazon Elastic Block Store (Amazon EBS)** proporciona almacenamiento persistente en bloques para utilizar con instancias de Amazon EC2.

Un volumen de Amazon EBS funciona de forma similar a un disco duro conectado a un servidor físico, permitiendo almacenar datos de forma persistente incluso cuando la instancia se detiene.

---

## Características principales

Amazon EBS ofrece almacenamiento diseñado para utilizarse junto con instancias Amazon EC2.

Sus principales características son:

- Almacenamiento persistente.
- Alto rendimiento y baja latencia.
- Posibilidad de crear instantáneas (*snapshots*).
- Capacidad de aumentar el tamaño del volumen.
- Integración con Amazon EC2.

---

## Persistencia de los datos

Los datos almacenados en un volumen Amazon EBS permanecen disponibles aunque la instancia EC2 se detenga.

Esto permite:

- Reiniciar una instancia sin perder la información almacenada.
- Sustituir una instancia conservando los datos del volumen.
- Restaurar información mediante instantáneas.

---

## Instantáneas (Snapshots)

Una **instantánea** es una copia de seguridad incremental de un volumen Amazon EBS almacenada en Amazon S3.

Las instantáneas permiten:

- Crear copias de seguridad.
- Restaurar un volumen.
- Crear nuevos volúmenes.
- Migrar datos entre zonas de disponibilidad.

---

## Casos de uso

Amazon EBS está especialmente indicado para:

- Discos del sistema operativo.
- Bases de datos.
- Aplicaciones empresariales.
- Sistemas que requieren baja latencia.
- Almacenamiento persistente para instancias EC2.

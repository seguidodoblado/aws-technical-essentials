# Objetivos

Al finalizar este módulo serás capaz de:

- Describir los principales servicios de computación de AWS.
- Explicar los casos de uso de Amazon EC2.
- Comprender el funcionamiento de Amazon EC2 Auto Scaling.
- Explicar el propósito de Elastic Load Balancing.
- Identificar los escenarios de uso de AWS Lambda.
- Diferenciar Amazon ECS, Amazon EKS y AWS Fargate.
- Describir las características de Amazon Elastic Block Store (Amazon EBS).

# Beneficios de las bases de datos en AWS

AWS ofrece servicios de bases de datos completamente administrados que ayudan a reducir la carga operativa y permiten centrarse en el desarrollo de las aplicaciones.

### Diseñadas para cada caso de uso

AWS proporciona más de quince servicios de bases de datos especialmente diseñados, entre los que se incluyen bases de datos relacionales, de clave-valor, documentales y en memoria.

---

### Rendimiento a escala

Los servicios de bases de datos de AWS están diseñados para proporcionar rendimiento, escalabilidad y disponibilidad con el fin de satisfacer las necesidades de aplicaciones que deben ejecutarse a cualquier escala.

---

### Completamente administradas

AWS se encarga de las tareas de administración de las bases de datos, como:

- Aprovisionamiento.
- Aplicación de parches.
- Configuración.
- Copias de seguridad.
- Recuperación.

---

### Seguras y de alta disponibilidad

Los servicios de bases de datos de AWS incorporan mecanismos de seguridad y permiten crear arquitecturas de alta disponibilidad para aumentar la resiliencia de las aplicaciones.

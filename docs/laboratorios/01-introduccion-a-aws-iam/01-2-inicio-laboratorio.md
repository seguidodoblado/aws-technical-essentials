# Inicio del laboratorio

Antes de comenzar las tareas del laboratorio, revisa los conocimientos previos recomendados, familiarízate con la simbología utilizada en la documentación y prepara el entorno de trabajo.

## Conocimientos técnicos previos (opcional)

Este laboratorio requiere:

- Un equipo con Microsoft Windows, macOS o Linux (Ubuntu, SUSE o Red Hat). Los dispositivos móviles no permiten visualizar correctamente la interfaz de usuario de la consola de AWS.
- Un navegador web:
  - Microsoft Windows: Microsoft Edge
  - macOS: Mozilla Firefox
  - Linux: Mozilla Firefox

## Iconografía

En el laboratorio se utilizan los siguientes iconos para identificar distintos tipos de instrucciones e información.

| Icono | Significado |
| :---: | ------------ |
| 📄 | **Resultado esperado.** Ejemplo de la salida que debería obtenerse al completar un paso o ejecutar un comando. |
| 📝 | **Nota.** Información adicional o instrucciones importantes. |
| ✔️ | **Tarea completada.** Indica la finalización de una tarea o un resumen de un procedimiento realizado en el laboratorio. |

## Preparación del laboratorio

1. En la parte superior de la página, selecciona **Start Lab**.

> **Precaución**
>
> Debes esperar a que todos los servicios de AWS se encuentren listos antes de poder utilizar la consola de AWS.

2. Abre la **AWS Management Console**.
3. Comprueba que la **región AWS** seleccionada sea **N. Virginia (us-east-1)**.

> **Advertencia**
>
> No cambies la región.

## Errores habituales al iniciar sesión

### Debes iniciar sesión primero

Si aparece el mensaje **You must first sign in**, significa que has abierto la consola de AWS antes de iniciar el laboratorio.

### Amazon Web Services Sign In

Si aparece la página **Amazon Web Services Sign In**, significa que ya existe una sesión iniciada con otra cuenta de AWS.

Para solucionarlo:

1. Selecciona el icono de usuario situado en la esquina superior derecha de la consola de AWS.
2. Selecciona **Sign Out**.
3. Cierra la página **Amazon Web Services Sign In** y vuelve a abrir la consola desde el laboratorio.

### Error: Choosing Start Lab has no effect

En algunos casos, al seleccionar **Start Lab**, puede parecer que no ocurre nada.

Las causas habituales son:

- El bloqueador de ventanas emergentes del navegador impide que se abra la pestaña **Start Lab**.
- El dominio del laboratorio está incluido en un bloqueador de anuncios o en una lista de filtros.

Actualiza la página e inténtalo de nuevo.

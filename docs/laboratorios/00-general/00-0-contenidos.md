# Contenidos

Este repositorio recopila los laboratorios del curso **AWS Technical Essentials**.

Cada laboratorio desarrolla un escenario práctico diferente y permite aplicar los conceptos estudiados durante el curso mediante ejercicios guiados.

---

## Laboratorios

- **Laboratorio 1. Introducción a AWS Identity and Access Management**
- **Laboratorio 2. Configuración de una aplicación web de alta disponibilidad en una instancia Amazon EC2**
- **Laboratorio 3. Configuración de una aplicación web en Amazon S3 y Amazon DynamoDB**
- **Laboratorio 4. Configuración de AWS Cloud9 para el laboratorio**

---

## Organización del repositorio

Cada laboratorio seguirá una estructura homogénea:

- Traducción completa del laboratorio.
- Explicación técnica de cada paso.
- Desglose de comandos, scripts y fragmentos de código.
- Capturas de la consola de AWS integradas en la documentación.
- Explicación del funcionamiento de los recursos creados durante el laboratorio.

El objetivo es que cada laboratorio pueda utilizarse tanto como guía para su ejecución como material de estudio para comprender el funcionamiento de los servicios de AWS empleados.
